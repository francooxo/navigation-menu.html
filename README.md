# 🧭 Semantic Navigation Menu - Day 2

This project is a clean, structural header layout built for my Day 2 web development challenge. It showcases how to design a modern navigation section.

---

## ⭐ Features
* **Semantic Structure:** Uses the clean HTML5 `<nav>` container element.
* **Branding Layout:** Places a site brand/logo link on the left side.
* **Navigation Links:** Features exactly 4 organized shortcuts (`Home`, `About`, `Services`, `Contact`) wrapped inside an unordered list.

## 🛠️ Built With
* HTML5
* CSS3
